# Adventures — Outdoor Block Playbook

Adventure blocks are the dashboard's name for movement-rich, hands-on learning. They're built by `buildAdventureBlock()` and always include a **materials list**, **numbered steps**, and an **outdoor option**.

## Why
- Ohio's IH statute counts physical education and applied learning toward instructional time.
- Reagan focuses better with at least one big-movement block per day.
- Outdoor adventures double as the day's gym block.

## Standing themes
- **Nature** — birding, leaf ID, weather observations.
- **Maker** — build, sew, cardboard engineering.
- **Movement** — geocaching, parkour drills, running games.
- **Community** — library trip, museum, farmer's market.

## Logging
On finish, tap the block → "Done". The day log auto-pulls the materials list + the outdoor option taken (or not). Photos taken in the moment can be attached via Adult Quick-Entry → Photo.

## Safety
- The outdoor option is opt-in; default is the indoor backup.
- Any block with `outdoor: true` requires Mom-tap "I'll be outside" before Reagan sees the GO button.
